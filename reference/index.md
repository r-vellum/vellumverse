# Package index

## Attaching the ecosystem

Loading `vellumverse` attaches the core packages. These helpers report
what was loaded and surface any masked functions.

- [`vellumverse_packages()`](https://r-vellum.github.io/vellumverse/reference/vellumverse_packages.md)
  : List the core vellum-ecosystem packages
- [`vellumverse_conflicts()`](https://r-vellum.github.io/vellumverse/reference/vellumverse_conflicts.md)
  : Conflicts between the vellum ecosystem and other packages
